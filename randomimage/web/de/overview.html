Überraschungsbild - Übersicht

Sie haben viele schöne digitale Fotos, aber keine Zeit, sie sich anzusehen? Dann brauchen Sie diese App!

Diese App ermöglicht es Ihnen, eine beliebige Anzahl von Bildern, die auf Ihrem Smartphone (oder Tablet, inkl. SD-Karte) gespeichert sind, in eine Liste zusammenzustellen, und sich dann, wann immer Sie möchten, ein zufälliges Bild aus dieser Liste anzeigen zu lassen.

Das klingt nicht wie eine große Sache, aber glauben Sie mir - es kann Ihnen viele glückliche Momente bringen - Sie müssen nur viele Freude bringende Bilder in Ihre Überraschungsbildliste legen!

Zum Beispiel können Sie sich jeden Tag überraschen lassen

  • Mit einem Foto von Ihrem/Ihrer Liebsten
  • Mit einem Foto von Ihrer letzten großen Urlaubsreise
  • Mit einem Foto Ihrer Kinder, als sie noch klein waren

Wählen Sie einfach viele Fotos, die Sie glücklich machen!

Funktionalitäten

Die App umfasst die folgenden Funktionen:

  • Bilder aus der Galerie in Ihre Überraschungsbildliste hinzufügen
  • Ein zufälliges Bild aus der Liste in Bildschirmgröße anzeigen
  • Vergrößern des angezeigten Bildes oder wechseln zu einem anderen Überraschungsbild
  • Ein Widget auf dem Startbildschirm, das ein regelmäßig wechselndes Überraschungsbild zeigt
  • Entfernen von Bildern aus der Liste

Wenn Ihnen die App gefällt, können Sie eine Premium-Erweiterung kaufen, das folgendes beinhaltet:

  • Anlegen mehrerer Bildlisten
  • Sichern und Wiederherstelln von Bildlisten
  • Gleichzeitige Verwendung von mehrere Widgets für verschiedene Bildlisten
