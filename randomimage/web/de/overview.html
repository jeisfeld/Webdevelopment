Zufallsbild - Übersicht

Sie haben viele schöne digitale Fotos, aber keine Zeit, sie sich anzusehen? Dann könnte diese App Ihnen helfen.

Diese App ermöglicht es Ihnen, eine beliebige Anzahl von Bildern und Verzeichnissen, die auf Ihrem Smartphone (oder Tablet, inkl. SD-Karte) gespeichert sind, in eine Liste zusammenzustellen, und sich dann, wann immer Sie möchten, ein zufälliges Bild aus dieser Liste anzeigen zu lassen.

Sie können verschiedene Listen anlegen, z.B.

  • Fotos von Ihrem/Ihrer Liebsten
  • Fotos von Ihren Urlaubsreisen
  • Fotos Ihrer Kinder, als sie noch klein waren

Anschließend können Sie sich die Fotos auf unterschiedliche Art anzeigen lassen:

  • Über ein Widget auf ihrem Startbildschirm, das regelmäßig das Bild wechselt
  • Über ein Startseitensymbol, das auf Anfrage ein zufälliges Bild Ihrer Liste anzeigt
  • Über Benachrichtigungen, die in wechselnden Intervallen erscheinen

Testen Sie die App und probieren Sie ihre Möglichkeiten aus!

Funktionalitäten

Die App umfasst die folgenden Funktionen:

  • Konfiguration von Zufallbildlisten
  • Anzeige eines zufälligen Bildes aus Ihrer Liste
  • Vergrößern des angezeigten Bildes oder Wechseln zu einem anderen Zufallsbild
  • Widgets auf dem Startbildschirm, die regelmäßig wechselnde Bilder aus Ihrer Liste zeigen
  • Konfiguration von Benachrichtigungen, die an zufälligen Zeitpunkten ein Bild aus Ihrer Liste anzeigen
  • Hinzufügen von „Teillisten“, die Sie unterschiedlich gewichten können (so dass z.B. seltene Bilder häufiger angezeigt werden)
  • Sichern und Wiederherstellen von Listen

Die App ist auf maximal drei Listen beschränkt. Wenn Ihnen die App gefällt, können Sie eine Premium-Erweiterung kaufen, die diese Beschränkung aufhebt (als separate App oder als In-App-Kauf).
