Hinweise zur Benutzung

Diese App zeigt zufällige Bilder von Bildlisten, die Sie konfigurieren. Daher müssen Sie zunächst eine Bildliste erstellen, bevor Sie die Anzeige zufälliger Bilder genießen können.

Erstellung der Bildliste

Erste Schritte

Wenn Sie die App zum ersten Mal öffnen, wird eine leere Bildeliste erstellt, und Sie sehen die Konfigurations-Seite für diese Liste. Auf dieser Seite können Sie das Symbol , drücken um Bilder zur Liste hinzuzufügen.

Hierfür haben Sie zwei Möglichkeiten:

  • Ein Bild-Verzeichnis hinzufügen: Dies ist der einfachste Weg, um viele Bilder in Ihre Liste zu bekommen. Wählen Sie einfach ein beliebiges Bild in der Galerie, und die App wird das gesamte Bildverzeichnis zu Ihrer Überraschungsbildliste hinzuzufügen.
  • Einzelne Bilder hinzufügen: Dies ist ein bisschen mühsamer, aber flexibler. Sie können einzelne Bilder aus der Galerie auswählen, die dann zur Überraschungsbildliste hinzugefügt werden. (Da die Galerie für jedes einzelne Bild neu geöffnet wird, sollten Sie eine Galerie-App als Standard für diese Aufgabe festlegen, falls Sie mehrere Galerie-Apps installiert haben.)

Nachdem Sie die Bilder auf diese Art zu Ihrer Liste hinzugefügt haben, können Sie alle Arten ausprobieren, ihre Bilder anzusehen.

Öffnen der Konfigurationsseite

Zu einem späteren Zeitpunkt können Sie weitere Bilder zu Ihrer Liste hinzufügen oder Bilder aus der Liste entfernen.

Um Ihre Bildliste zu konfigurieren, müssen Sie generell die Konfigurationsseite für die Bildliste öffnen. Dies können Sie auf folgende Arten tun:

  • Durch zweimaliges Antippen eines Bildes, das sie in der App sehen.
  • Durch langes Drücken auf ein Bild, während Sie es sehen, und anschließendes Drücken von „Bildliste anzeigen“.
  • Durch das Hinzufügen des Symbols für die Überraschungsbild-Konfiguration auf Ihre Startseite. (Dies ist ein zweites Startsymbol der App, neben dem Standard-Symbol, welches ein Überraschungsbild aufruft.)

Hinzufügen von Bildern auf die Überraschungsbildliste

Wenn Sie Bilder zu Ihrer Liste hinzufügen wollen, gibt es die folgenden Möglichkeiten:

  • Drücken Sie auf der Bildlisten-Konfigurationsseite auf das Symbol - dann können Sie wie beim ersten Mal Verzeichnisse oder Bilder zu Ihrer Bildliste hinzuzufügen. Dies ist der einzige Weg, wie Sie ganze Verzeichnisse hinzufügen können.
  • Alternativ können Sie ein Foto in der Galerie-App anzeigen es von dort an die Überraschungsbild-App senden.
  • Sie können auch mehrere Bilder in der Galerie App auswählen diese auf einmal an die Überraschungsbild-App senden. Dies ist der einfachste Weg, um viele einzelne Bilder aus dem gleichen Ordner zur Bildliste hinzuzufügen.

Entfernen von Bildern aus der Überraschungsbildliste

Wenn Sie Bilder aus der Liste entfernen möchten, gibt es folgende Möglichkeiten:

  • Drücken Sie auf der Bildlisten-Konfigurationsseite, auf das Symbol . Dann wählen Sie die Ordner und Bilder, die Sie aus der Liste entfernen möchten. Drücken Sie dann das Symbol um sie zu entfernen.
  • Hier können Sie auch das Symbol drücken, um alle Ordner und Bilder in der Liste auszuwählen. Dies ist hilfreich, wenn Sie den Großteil der Bilder entfernen wollen.
  • Während Sie ein Bild anschauen, können Sie das Bild aus der Liste entfernen, indem Sie zunächst lange auf das Bild drücken und dann „Bild aus der Liste entfernen“ wählen. Dies ist nur möglich für Bilder, die individuell in die Liste aufgenommen wurden - Verzeichnisse bleiben immer als Ganzes in der Liste.

Bilder ansehen

Anschauen zufälliger Bilder über das Startseitensymbol

  • Wenn Sie das Startseiten-Symbol drücken, wird die App ein zufälliges Bild aus Ihrer Überraschungsbildliste im Vollbild anzeigen.
  • Wenn Sie mehr Details sehen wollen, können Sie es wie üblich mit zwei Fingern vergrößern.
  • Wenn Sie ein anderes Bild aus der Liste sehen wollen, dann schnipsen Sie das Bild einfach weg.
  • Wenn Sie wissen wollen, wo die Bilddatei liegt, drücken Sie etwas länger auf das Bild.

Verwenden von Widgets

Es gibt drei Arten von Widgets, die Sie auf Ihrem Home-Bildschirm platzieren können, um zufällige Bilder zu sehen.
  • Überraschungsbild-Widget: Dieses Widget verhält sich wie der das Startseitensymbol, solange Sie nur eine Bildliste haben. Wenn Sie mehrere Bildlisten konfiguriert haben, dann ruft man mit dem Überraschungsbild-Widget ein zufälliges Bild einer bestimmten Bildliste auf.
  • Überraschungsbilderrahmen: Dieses Widget legt eines der Bilder Ihrer Bildliste auf dem Startbildschirm. Beim Ablegen des Widgets auf den Startbildschirm können Sie angeben, wie oft eine neues Überraschungsbild angezeigt werden soll.
  • Überraschungsbilderstapel: Dieses Widget legt die Bilder Ihrer Liste in zufälliger Reihenfolge als Kartenstapel auf dem Home-Bildschirm. Sie können ssich von einem Bild zum nächsten bewegen. Wenn Sie das Widget auf dem Startbildschirm ablegen, können Sie angeben, wie oft der Stapel neu gemischt werden soll.

Wenn Sie auf den Überraschungsbilderrahmen oder den Überraschungsbilderstapel tippen, dann können Sie das angezeigte Foto im Detail anschauen. Von hier aus können das Bild wegschnippen, um weitere Bilder der gleichen Liste zu sehen, oder Sie können die Konfigurationsseite für diese Liste öffnen.

Wenn Sie mehrere Bildlisten haben (wozu Sie die Premium-Erweiterung benötigen), dann können Sie auch Widgets für mehrere Bildlisten auf Ihren Startbildschirm legen, so dass Sie zum Beispiel immer ein Foto von Ihrem Partner und ein Foto von Ihrem letzten Urlaub auf dem Bildschirm haben können.

Bitte beachten Sie, dass direkt nach einem Neustart des Telefons oder Tablets die externe SD-Karte noch nicht verfügbar ist. Daher können nach einem Neustart nur Bilder aus dem internen Telefonspeicher in den Widgets angezeigt werden. Wenn Sie auch direkt nach Neustarts ein schönes Bild sehen wollen, sollten Sie einen nettes Foto aus dem internen Telefonspeicher in jede Ihrer Bildlisten aufnehmen.

Die Widgets können das angezeigte Bil häufiger wechseln als angegeben (abhängig vom Android-System).

Nutzung mehrerer Bildlisten (Premium)

Über das Symbol können Sie das Menü für die Konfiguration mehrerer Bildlisten öffnen. Es enthält die folgenden Einträge:

  • Bildliste wechseln: Sobald Sie mehr als eine Bildliste haben, können Sie diesen Menüpunkt auswählen, um zur Konfigurationsseite einer anderen Bildliste zu gelangen. Die ausgewählte Liste wird dann auch vom Startseitensymbol aus verwendet
  • Neue Bildliste erstellen: Hier können Sie eine neue leere Bildliste erstellen
  • Aktuelle Bildliste duplizieren: Hier können Sie eine neue Bildliste erstellen, die aus der aktuellen Liste vorbefüllt ist.
  • Aktuelle Bildliste umbenennen: Hier können Sie den Namen der aktuellen Bildliste ändern.
  • Bildliste sichern: Hier können Sie ein Backup einer Ihrer Bilder-Listen erstellen. Das Backup wird im Verzeichnis „RandomImage“ Ihres Telefonspeichers abgelegt, von wo aus Sie es an jeden anderen Ort kopieren können.
  • Bildliste wiederherstellen: Hier können Sie eine beliebige Bildliste, deren Sicherung sich im Verzeichnis „RandomImage“ Ihres Telefonspeichers befindet, wiederherstellen.
  • Bildliste löschen: Hier können Sie eine beliebige Bildiste (mit Ausnahme der aktuellen Liste) löschen. Diese wird permanent gelöscht, man kann sie aber wiederherstellen, falls Sie ein Backup haben.

Ohne Premium-Erweiterung wird durch das Symbol das Einstellungs-Menü geöffnet.

Weitere Einstellungen

Während Sie sich auf der Bildlisten-Konfigurationsseite befinden, können Sie auf das Symbol drücken, um zur Einstellungsseite zu gelangen. Hier finden Sie folgende Kategorien von Einstellungen:

Premium-Erweiterungen

Sie können Sie Premium-Erweiterungen erwerben, die die Verwendung von mehreren Bildlisten in der App ermöglichen.

Derzeit sind beide Premium-Erweiterungen gleichwertig. Die Premium-Plus-Erweiterung drückt nur zusätzliche Anerkennung an den Entwickler der App aus.

Auf lange Sicht wird die Premium-Plus-Erweiterung immer alle Funktionen der App freischalten. Voraussichtlich werden neue Funktionen kommen, die nicht durch die Premium-Erweiterung aktiviert sind. Daher ist die Premium-Plus-Erweiterung eine Art Flatrate für zukünftige App-Erweiterungen.

Unterstützung

Hier können Sie diese Hilfe-Seite öffnen, eine PayPal-Spende an den Entwickler machen, oder den Entwickler per E-Mail kontaktieren.

Weitere Einstellungen

  • Sprache: Hier können Sie die Sprache der App ändern. (Derzeit unterstützt: Englisch und Deutsch)
  • Alle Hinweise zeigen: Es gibt verschiedene Informations-Dialoge in der App, die Sie durch Drücken der Taste „Nicht mehr zeigen“ permanent deaktivieren können. Die Schaltfläche „Alle Hinweise zeigen“ aktiviert all diese Nachrichten wieder.
  • Hinweise nicht mehr zeigen: Dies deaktiviert alle Arten von Info-Meldungen, die sonst individuell ausgeschaltet werden müssen. Dies ist hilfreich für erfahrene Nutzer, die die Anwendung auf einem Gerät installiert haben.
