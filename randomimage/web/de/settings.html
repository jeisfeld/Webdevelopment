Einstellungen

App-Einstellungen

Während Sie sich auf der Haupt-Konfigurationsseite befinden, können Sie auf das Symbol drücken, um zur Einstellungsseite zu gelangen. Hier finden Sie folgende Kategorien von Einstellungen:

Premium-Erweiterungen

Sie können Sie Premium-Erweiterung erwerben, die die Verwendung von mehr als drei Bildlisten in der App ermöglichen.

Alternativ können Sie die App „Überraschungsbild Pro“ erwerben.

Unterstützung

Hier können Sie diese Hilfe-Seite öffnen, eine PayPal-Spende an den Entwickler machen, oder den Entwickler per E-Mail kontaktieren.

Weitere Einstellungen

  • Verzeichnisauswahl: Hier können Sie festlegen, auf welche Art Sie Bilder und Verzeichnisse zum Hinzufügen auswählen:
    • Liste aller Bildverzeichnisse (Voreinstellung): Hier wird Ihnen die Liste aller Bildverzeichnisse angezeigt, und Sie können aus dieser Liste auswählen. Dies ist die einfachste Option.
    • Dateibrowser: Hier können Sie Ihre Bildverzeichnisse in einem Dateibrowser auswählen.
    • Galerie: Hier können Sie die Bildverzeichnisse in der Galerie-App auswählen. Dies kann hilfreich sein, wenn Sie Ihre Bilder in der Galerie am einfachsten finden.
  • Sprache: Hier können Sie die Sprache der App ändern. (Derzeit unterstützt: Englisch und Deutsch)
  • Benachrichtigung bei Listenänderung: Hier können Sie festlegen, dass eine Benachrichtigung angezeigt wird, wenn Sie etwas zu einer Bildliste hinzufügen oder entfernen.
  • Regexp-Filter verwenden: Dies erlaubt, gewisse Listen oder Verzeichnisse generell aus der Anzeige herauszufiltern.
  • Alle Hinweise zeigen: Es gibt verschiedene Informations-Dialoge in der App, die Sie durch Drücken der Taste „Nicht mehr zeigen“ permanent deaktivieren können. Die Schaltfläche „Alle Hinweise zeigen“ aktiviert all diese Nachrichten wieder.
  • Hinweise nicht mehr zeigen: Dies deaktiviert alle Arten von Info-Meldungen, die sonst individuell ausgeschaltet werden müssen. Dies ist hilfreich für erfahrene Nutzer, die die Anwendung auf einem Gerät installiert haben.

Widget-Einstellungen

  • Listenname: Hier können Sie einstellen, welche Bildliste das Widget verwenden soll.
  • Bildliste bearbeiten: Dieser Link verweist zur Bearbeitung der Bildliste, für die das Widget konfiguriert ist.
  • Wechselhäufigkeit: Hier können Sie einstellen, wie häufig das Bild wechseln soll.
  • Als Liste anzeigen: Beim Überraschungsbilderstapel kann hier eingestellt werden, dass die Bilder nicht als Stapel sondern als Liste angezeigt werden.
  • Alle Bilder zyklisch anzeigen: Beim Überraschungsbilderstapel kann hier eingestellt werden, ob hier alle Bilder der Liste in den Stapel kommen, oder ob anhand der Gewichtungen nur einige Bilder aufgenommen werden (was bei sehr großen Bildersammlungen empfehlenswert ist).
  • Hintergrund: Hier können Sie die Hintergrundfarbe des Widgets festlegen. Die Auswahl „Rahmen mit Bild füllen“ bedeutet, dass das Bild gestreckt wird, um das ganze Widget auszufüllen.
  • Button-Stil: Hier können Sie einstellen, wo die zwei Widget-Buttons angezeigt werden. Dies sind:
    • Ein Button zum ändern der Settings
    • Ein Button, um das angezeigte Bild zu wechseln
  • Button-Farbe: Hier können Sie einstellen, welche Farbe diese beiden Buttons haben sollen.
  • Name des Widgets: Hier können Sie einen Namen angeben, der in der Widget-Konfiguration angezeigt wird.
  • Skalierung in Detailansicht: Hier können Sie einstellen, wie die Fotos skaliert werden sollen, wenn Sie durch Drücken auf das Widget zur Detailansicht wechseln.
  • Hintergrund in Detailansicht: Hier können Sie einstellen, welche Hintergrundfarbe angezeigt werden soll, wenn Sie durch Drücken auf das Widget zur Detailansicht wechseln.
  • Nutzungsdauer des Widgets: Hier können Sie beschränken, wie lange Sie Bilder über das Mini-Widget anschauen können.
  • Sperrdauer des Widgets: Hier können Sie beschränken, wie oft sie das Mini-Widget nutzen können.

Benachrichtigungs-Einstellungen

  • Listenname: Hier können Sie einstellen, welche Bildliste die Benachrichtigung verwenden soll.
  • Bildliste bearbeiten: Dieser Link verweist zur Bearbeitung der Bildliste, für die die Benachrichtigung konfiguriert ist.
  • Häufigkeit der Benachrichtigung: Hier können Sie einstellen, wie häufig eine Benachrichtigung angezeigt werden soll.
  • Variabilität der Benachrichtigung: Hier können Sie wählen, ob die Benachrichtigungen ungefähr in gleichen Abständen oder in größerer Variabilität kommen sollen.
  • Startzeit der Benachricheigungen, Endzeit der Benachrichtigungen: Hier können Sie ein Zeitfenster angeben, in dem die Benachrichtigungen kommen können.
  • Anzeigedauer: Hier können Sie festlegen, die Benachrichtigungen nach einer gewissen Zeit automatisch zu löschen.
  • Benachrichtigungsstil: Hier können Sie zwischen drei Arten von Benachrichtigungen wählen.
  • LED-Farbe: Hier können Sie festlegen, ob die LED zur Benachrichtigung blinkt.
  • Vibration: Hier können Sie festlegen, ob das Gerät bei der Benachrichtigung vibriert.
  • Farbiges Symbol: Hier können Sie festlegen, ob das Benachrichtigungssymbol farbig oder weiß sein soll.
  • Name der Benachrichtigung: Hier können Sie einen Namen angeben, der in der Benachrichtigungs-Konfiguration angezeigt wird.
  • Skalierung in Detailansicht: Hier können Sie einstellen, wie die Fotos skaliert werden sollen, wenn Sie durch Drücken auf die Benachrichtigung zur Detailansicht wechseln.
  • Hintergrund in Detailansicht: Hier können Sie einstellen, welche Hintergrundfarbe angezeigt werden soll, wenn Sie durch Drücken auf die Benachrichtigung zur Detailansicht wechseln.
  • Verhalten beim Schnippen: Hier können Sie angeben, ob beim Schnippen wie üblich ein neues Bild angezeigt wird, oder ob stattdessen die Nachricht beendet wird.
  • Benachrichtigung entfernen: Hier können Sie die Benachrichtigung entfernen.
