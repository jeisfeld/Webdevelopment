Release Notes

  • Version 0.2.4: In case of list images that cannot be found, display message only while organizing, not while viewing.
  • Version 0.2.3: Offered more possibilities to add images to list.
  • Version 0.2.2: Simplified the first steps.
  • Version 0.2.1: Possibility to individually adjust the frequency of picture selection for each nested list.
  • Version 0.2:
    • Quick adding of folders via long click on folder name.
    • Adding of image lists into other image lists.
    • Possibility to send image (e.g. per mail)
    • Bugfixes.
  • Version 0.1.7: Selection of folders directly out of the app.
  • Version 0.1.6: Improved mechanism to add images.
  • Version 0.1.5:
    • Surprise Image Frame: added buttons to switch to another image or to change widget settings.
    • List Setup: enabled long press to show details.
    • Bugfixes.
  • Version 0.1.4: Flinging enhancements.
    • Enable to fling back to the previously shown image.
    • Improved flinging performance and flinging animation.
  • Version 0.1.3: Changed name from "Random Image" to "Surprise Image". Changed icon background from orange to blue.
  • Version 0.1.2: Bugfixes.
  • Version 0.1.1: Initial version of the app.
    • Display random image, and change image by flinging.
    • Add images and folders to image list via gallery or via "send".
    • Delete images and folders from image list.
    • Manage multiple image lists (premium only).
    • Three kinds of widgets
    • Display image information of the current image.
    • Select language (English or German).
