How to use

This app displays surprise images from image lists that you configure. Therefore, you first have to configure an image list before you can enjoy the display of surprise images.

Setup of the image list

First steps

When you open the app for the first time, an empty image list is created, and you see the setup page for this list. On this page, you may press the icon in order to add images to the list. Select the folder from which you want to add images to the list. Then the list of images in this folder gets displayed, and you have the following alternatives:

  • Add the whole folder: This is the easiest way to get many images into your list. To do this, press the icon .
  • Add individual images: This is a bit more laborious, but more flexible. Just select all images that you want to add, and then press the icon

After you have added folders and images to your list in this way, you can try all variants to view the images.

Opening the configuration page

At a later time, you may want to add further images to your list, or to remove images from your list.

Generally, in order to configure your image list, you have to open the configuration page of the list. You can access this page in the following ways:

  • By double tapping on an image while viewing it.
  • By long pressing on an image while viewing it, and then pressing “Display image list”.
  • By adding the launcher for the surprise image configuration to your home screen. (This is a second launcher provided by the app.)

Adding images to the surprise image list

If you want to add images to your list, there are the following options:

  • On the image list configuration page, press the icon - then you can add folders or images as in the beginning. This is the only way in which you can add whole folders.
  • While browsing the folders, you may make a long press on a folder name - this allows to add this folder quickly to the image list without looking into its contents. This is helpful if you want to add many folders.
  • Alternatively, you may display a photo in the gallery app and send the image from there to the Surprise Image app.
  • Or, you may select multiple images in the gallery app and send them at once to the Surprise Image app. This is the easiest way to add many individual images from the same folder.

Removing images from the surprise image list

If you want to remove images from your list, there are the following options:

  • On the image list configuration page, press the icon . Then select the folders and images that you want to remove from the list. Then press the icon to remove them.
  • Here, you may also press the icon to select all folders and images in the list. This is helpful if you want to remove the majority of images.
  • Alternatively, while viewing an image, you may long press on the image, and then select “Remove image from list” in order to remove the image that you are viewing. This is possible only for images that are added individually to the image list - folders always stay as a whole in the list.

View the images

Viewing surprise images via the launcher

  • If you press the launcher icon, the app will start displaying a random image from your surprise image list on full screen.
  • If you want to see more details, just zoom as usual with two fingers.
  • If you want to see a different image from the list, just flip the image away.
  • You can go one image back by flipping in the other direction.
  • If you want to know the file location of this image, make a long press on the image. From here, you may also send the image (e.g. via email).

Using widgets

There are three types of widgets that you can place on your home screen in order to see surprise images.
  • Surprise Image Widget: This widget behaves like the launcher, as long as you have only one image list. If you have multiple image lists configured, then the Surprise Image Widget allows to display a surprise image of a specific image list.
  • Surprise Image Frame: This widget places one of the images of your list on the home screen. When placing the widget on your home screen, you can specify how frequently a new surprise image should be displayed.
  • Surprise Image Deck: This widget places the images of your list in random order as a card deck on the home screen. You can move from one image to the next. When placing the widget on your home screen, you can specify how frequently the pile should be re-shuffled.

If you tap on the Surprise Image Frame or the Surprise Image Deck, then you can see the displayed photo in detail. From here, you may flip the image away in order to see other images from the same list, or you may open the configuration page for this list.

The Surprise Image Frame also allows you to switch to a new random image (via tap on the right border of the frame), or to change the configuration of the frame (via tap on the left border of the frame).

If you have multiple image lists (which requires premium add-on), then you can place widgets for multiple image lists on your home screen, so for example you may have always a photo of your spouse and a photo of your last vacation in parallel on your screen.

Please note that directly after a reboot of the phone or tablet, the external SD card is not yet accessible. Therefore, after a reboot, the widgets can only display images from internal phone memory. If you want to see a nice image directly after reboot, you should add one nice photo from internal phone memory to each of your image lists.

The widgets may update the image more frequently than specified (dependent on the Android system).

Use of multiple image lists (Premium)

Via the icon you can access the menu for handling multiple image lists. It contains the following entries:

  • Switch image list: As soon as you have more than one image list, you can use this menu item to select another image list in the configuration. The selected list will also be used by the launcher.
  • Create new image list: Here you can create a new empty image list.
  • Clone current image list: Here you can create a new image list which is pre-filled from the current list.
  • Rename current image list: Here you can change the name of the current image list.
  • Backup image list: Here you can backup any of your image lists. The backup is stored in the folder “RandomImage” of your phone memory, from where you may copy it to any other place.
  • Restore image list: Here you can restore any image list whose backup is located in the folder “RandomImage” of your phone memory.
  • Delete image list: Here you can delete any image list (except the current one). This is a permanent deletion, but you may restore it if you have a backup.
  • Include other list in current list: Here you can another image list to the current image list. This gives the possibility to combine multiple image lists into one bigger image list.

Included lists are displayed as a blue folder. If you make a long click on such a list, you may adjust how frequent images from this nested list are taken. In this way you may increase the display frequency of pictures from small lists.

Without premium add-on, the icon opens the setup menu.

Additional settings

While being on the image list configuration page, you can press the icon to go to the settings page. Here you find the following categories of settings:

Premium add-ons

You can purchase premium add-ons that enable the use of multiple image lists in the app.

Currently, both premium add-ons are equivalent. The “Premium plus” add-on just gives additional appreciation to the developer of the app.

On the long run, the “Premium plus” add-on will always enable all features of the app. Presumably, there will come new features which are not enabled by the “Premium” add-on. Therefore, the “Premium plus” add-on is a kind of flatrate for future app extensions.

Support

Here you can open this help page, make a PayPal donation to the developer, or contact the developer via email.

Other settings

  • Folder selection: Here you can select between three different mechanisms to add images and folders:
    • List of image folders (default): Here you get the list of all image folders displayed and can select from this list. This is the easiest option if you have few image folders.
    • File browser: Here you can select your image folders in a file browser. This is most practical if you have many image folders organized in the file system.
    • Gallery: Here you can select the image folder via the Gallery app. This may be helpful if you find your images easiest in the gallery.
  • Language: Change the language of the app. (Currently supported: English and German)
  • Display all info messages: There are various info messages in the app which you may disable by pressing the button “Do not show again”. The button “Display all info messages” re-enables all these messages.
  • Do not display info messages: This disables all kinds of info messages which otherwise need to be switched off individually. This is helpful for experienced users who install the app on a new device.
